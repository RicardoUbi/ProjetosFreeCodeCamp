# Calculadora Financeira

Este projeto visa o desenvolvimento de uma calculadora financeira completa utilizando a linguagem de programação Python. A calculadora deve oferecer diversas funcionalidades para auxiliar usuários em cálculos financeiros pessoais e profissionais.

Principais operações que a calculadora realiza:

- Cálculo de Anuidade: A calculadora deve calcular o valor futuro de uma anuidade, considerando crescimento mensal ou contínuo.

- Cálculo de Parcela Mensal de Hipoteca: A calculadora deve determinar o valor da parcela mensal necessária para quitar um empréstimo imobiliário (hipoteca), considerando o valor principal do empréstimo, a taxa de juros anual e o prazo total do financiamento.

- Estimativa de Saldo de Investimento para Aposentadoria: A calculadora deve estimar o saldo futuro de um investimento para aposentadoria, levando em conta fatores como contribuições periódicas, taxa de juros e tempo de investimento até a aposentadoria.

-  Estimativa para tempo de duplicação de um investimento: A calculadora deve determinar o tempo necessário para que um valor inicial dobre a uma determinada taxa de juros anual.

- Resolução de Equações Logarítmicas: A calculadora deve resolver equações que envolvam funções logarítmicas.

- Conversão para Notação Científica: A calculadora deve converter números para a notação científica (formato exponencial).

Ao concluir este projeto, o usuário terá desenvolvido uma ferramenta prática para realizar diversos cálculos financeiros, auxiliando no planejamento financeiro, tomada de decisões de investimento, estimativa de aposentadoria e resolução de problemas financeiros. A calculadora financeira em Python torna-se um recurso valioso para pessoas físicas, profissionais financeiros e estudantes que buscam aprimorar seus conhecimentos em finanças.
